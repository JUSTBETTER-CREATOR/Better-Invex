# Publicar Better - Invex en GitHub Pages

## Nombre sugerido

`Better-Invex`

## Crear el repositorio

1. En GitHub, presiona `+` y selecciona `New repository`.
2. Escribe `Better-Invex`.
3. Selecciona `Public`.
4. No agregues README, `.gitignore` ni licencia porque esta carpeta ya los incluye.
5. Presiona `Create repository`.
6. Copia la dirección HTTPS, parecida a `https://github.com/TU-USUARIO/Better-Invex.git`.
7. Ejecuta `1_CREAR_REPO_Y_SUBIR.bat` y pega esa dirección.

## Activar GitHub Pages

1. Abre `Settings`.
2. Entra a `Pages`.
3. En `Build and deployment` selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
4. Presiona `Save`.

La dirección quedará parecida a:

`https://TU-USUARIO.github.io/Better-Invex/`

## Subir cambios después

Cada vez que modifiques la página, ejecuta `2_SUBIR_CAMBIOS.bat`.

## Contenido

- Postales: `assets/postales/`
- PDF, Word, Excel y formatos: `assets/documentos/`
